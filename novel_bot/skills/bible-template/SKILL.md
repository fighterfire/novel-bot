# 故事圣经

本目录存储故事的核心设定和追踪信息。

## 目录结构

```
bible/
├── characters/          # 角色档案
│   ├── main/           # 主角
│   ├── supporting/     # 配角
│   └── minor/          # 次要角色
├── plot/               # 情节追踪
│   ├── outline.md      # 大纲
│   ├── suspense.md     # 悬念追踪
│   └── foreshadow.md   # 伏笔追踪
├── world/              # 世界观
│   ├── setting.md      # 基本设定
│   ├── locations.md    # 地点
│   └── rules.md        # 规则体系
└── changelog.md        # 更新日志
```

## 使用说明

- 角色档案记录角色的基本信息、性格、关系、状态变化
- 情节追踪记录悬念和伏笔的埋设与回收
- 世界观记录故事的背景设定
- 更新日志记录每次重要变更
